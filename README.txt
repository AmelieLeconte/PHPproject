NOM : Leconte 
PRENOM : Amélie 
THEME : capitalisme

Ce projet fait honneur au capitalisme et constitue mon premier rendu du module CSC4101. Tout est fonctionnel jusqu'au TP7. Il s'agit d'exposer des COINS dans des BANKS. Ces entitées sont manipulés grace à leurs controlleurs respectif. 

BANK possède l'attribut name en plus de id.
COIN possède l'attribut value en plus de id.

Une seule banque a été crée "Gringotts" contenant plusieurs coins : ceux de Paul, Agathe, Thomas et Augustin.

Le projet prends forme grâce à des fichiers twig et CSS afin d'ajouter des headers et des footers, manipulables dans le dossier template. Les fichiers SHOW permettent l'affichage d'un objet seul, tandis que les fichiers BASE permettent l'affichage des listes.
